How to Use BBC StoryPlayer

StoryPlayer looks for Stories in a folder named 'storyplayer', inside your My Documents folder.

Create a folder inside 'storyplayer' for each Story, and add the story.json file and associated media files to it.

All stories that StoryPlayer can find are listed above. Clicking Rescan Folder will rescan the storyplayer folder.
