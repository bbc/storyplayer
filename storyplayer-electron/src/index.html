StoryPlayer

How to Use this Application

On install if the application doesn't create a folder 'storyplayer' in your 'My Documents' you will have to do so manually

The application will read from your my Documents folder, on Windows this is your 'My Documents folder' On MacOS this is simply 'Documents'

Any story you wish to view can be added to this folder. Please ensure the story data model and the media are in the same folder.
