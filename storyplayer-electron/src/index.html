StoryPlayer

How to Use this Application

On install if the application doesn't create a folder 'storyplayer' in your My documents folder. If the application does not create the folder, you will have to do so manually.

The application will read from your storyplayer folder

Any story you wish to view can be added to this folder as separate folders for each story. Please ensure the story data model .json file and the corresponding media are in the same folder.
